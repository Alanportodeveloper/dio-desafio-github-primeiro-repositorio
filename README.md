Com as aulas de Git e Github, pude aprender:

## Comandos básicos-

 . Git init;
 . Git add;
 . Git commit;
 . Git status;
 . Git remote;
 . Git remote -V;
 . Git push;
 . Git Pull.

## Configurações básicas;

 . Git config --Global;
 . Criar chves SSh e TOKEN.

 . Git help;

 . Git diff;

 . Git checkout <commit>;
 . Git checkout master;
 . Git checkout -- <path_or_file>;
 . Git checkout HEAD -- <path_file>;
 . Git revert <commit>;
 . Git reset <commit>;
 . Git reset --hard <commit>
 . Git branch;
 . Git branch -d;
 . Git checkout <branch>;
 . Git merge <branch>;
 . Git rebase <branch>;
 . Git fetch;
 . Git tag;
 . Git fetch origin pull/ID/HEAD:BRANCH;
 . Git commit -ammend;
 . Git stash;
 . Git stash pop;
 . Git cerrypick <commit>;
 . Git blame;
 . Git bisect.
